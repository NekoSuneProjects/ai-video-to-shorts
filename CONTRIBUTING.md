# Contributing

Thanks for helping improve Shorts Lab. Contributions are welcome!

## Quick start

1. Fork the repo
2. Create a feature branch
3. Make changes
4. Open a pull request

## Development setup

```bash
npm install
npm run dev
```

## What to work on

- Highlight detection (audio energy / reactions)
- Better caption alignment and styles
- More export presets (TikTok, Reels)
- UI polish and accessibility
- Performance improvements

## Code style

- Keep changes focused and small
- Prefer readable code over clever tricks
- Add comments only when needed
- Test your change when possible

## Reporting bugs

Please include:
- OS and hardware
- Steps to reproduce
- Expected vs. actual behavior
- Logs or error messages

## Security

If you find a security issue, please open a private issue or email the maintainer first.
